# webxr-boilerplate
WebXR basic setup with AR and VR examples

# Setup

## ViteJS
Dev environment uses [ViteJS](https://vitejs.dev/).

### Test and serve app
```
npm run dev
```

### Build site for prod
```
npm run build
```
Compiles the site to the `dist` directory.

## main.js

Config the `XRtype` on line 6 of `main.js` to create a VR or AR app.
